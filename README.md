Este es un programa escrito en Go que analiza una contraseña ingresada por el usuario y evalúa su seguridad.

Características

El programa revisa si la contraseña contiene:

Letras mayúsculas

Letras minúsculas

Números

Signos de puntuación

También calcula un puntaje de seguridad dependiendo de los elementos que tenga la contraseña.

Cómo ejecutar el programa

Tener instalado Go

Ejecutar el archivo:

go run validaR.go

Ingresar una contraseña cuando el programa lo solicite.

Ejemplo

Entrada:

Hola123!

Salida:

Tiene mayúsculas
Tiene minúsculas
Tiene números
Tiene signos
Puntaje de seguridad: 4
Contraseña fuerte
